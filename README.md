# PPEUM Global · 관리자 콘솔 (web)

PPEUM Global API(`../api`)를 사용하는 사용자/관리자용 프론트엔드입니다.
React + TypeScript + Vite 기반이며, 로그인부터 상품·이벤트·시술 설명·이미지까지
실제 데이터 CRUD가 가능합니다.

## 보안 / 깃 안전성

- 자격증명·토큰을 코드에 하드코딩하지 않습니다. 로그인 후 발급된 토큰은 브라우저
  `localStorage` 에만 저장됩니다.
- API 주소는 `VITE_API_BASE_URL` 환경변수로만 주입합니다. 실제 `.env` 는 `.gitignore`
  처리되어 있고, 값이 없는 `.env.example` 만 커밋됩니다.
- AWS 키 등 백엔드 시크릿은 이 프로젝트에 포함되지 않습니다(서버 `.env` 에만 존재).

## 시작하기

```bash
# 1) 환경변수 준비
cp .env.example .env        # 필요 시 VITE_API_BASE_URL, VITE_PORT 수정

# 2) 의존성 설치 (이미 설치되어 있으면 생략)
npm install

# 3) 개발 서버 실행
npm run dev                 # 기본 http://localhost:5173
```

API 서버가 `http://localhost:3000` 에서 실행 중이어야 합니다. 다른 주소라면 `.env` 의
`VITE_API_BASE_URL` 을 수정하세요. (서버 측 CORS 가 활성화되어 있어야 합니다.)

## 환경변수

| 변수 | 설명 | 기본값 |
| --- | --- | --- |
| `VITE_API_BASE_URL` | API 서버 주소 | `http://localhost:3000` |
| `VITE_PORT` | 개발 서버 포트 | `5173` |

## 화면 구성 / 플로우

1. **로그인** (`/login`) — 어드민/유저 선택 후 아이디·비밀번호 로그인 → 토큰 저장 → 대시보드 이동.
   미인증 상태로 보호 라우트 접근 시 자동으로 로그인 화면으로 이동합니다.
2. **대시보드** (`/`) — 상품/이벤트/시술 설명/이미지 건수 요약.
3. **상품** (`/products`) — 목록·검색, 등록/수정(`/products/:id`).
   기준 언어(일본어) 기본 정보 + 분류/가격/노출/시술 연결/메뉴판 노출, 언어 탭에서
   공용(한국어)·기타 언어 번역 편집(기준 원문 및 재검수 항목 표시).
4. **상품 분류** (`/categories`) — 대분류/중분류 2단 구성, 다국어 명칭 일괄 편집.
5. **이벤트** (`/events`) — 목록, 등록/수정(`/events/:id`), 색상/기간/요일, 다국어 번역,
   연결 상품(이벤트가·할인율·순서) 관리.
6. **메뉴판** (`/menu-board`) — 분류 선택 후 상품 노출 순서 변경/저장.
7. **시술 설명** (`/operation-info`) — 목록, 등록/수정(`/operation-info/:id`), 요약 정보·해시태그·
   주의사항·이미지, 언어 탭 번역 편집.
8. **이미지** (`/images`) — 분류별 목록, 업로드/수정(멀티파트)·삭제. 다른 화면의 이미지 선택에 재사용.

상단 헤더의 **표시 언어** 선택값이 모든 요청의 `lang` 헤더로 전송되어, 목록·상세가 해당
언어 기준으로 표시됩니다. (이 프로젝트는 기준 언어·공용 언어가 모두 한국어 `ko` 입니다.)

```
  lib/          api 클라이언트, 인증/언어/토스트 컨텍스트, 타입·상수·조회 헬퍼

```
